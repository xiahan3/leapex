## 四、建议的端到端测试走法
1. **初始化**：录入「公司资料」+ 导入上表「期初试算表」→ 校验借贷平衡（1,075,000）通过。
2. **本期做账**：上传银行流水 / 票据（可用 AI 基于对账单造的测试票据），AI 匹配生成分录；其中**留一笔 USD 收款 / 付款**，验证按月平均汇率入账。
3. **期末调整**：计提季度折旧（汽车 / 办公 / 电脑）、应计核数费；**外币货币性项目按 2026-03-31 期末汇率重估**（如 USD 户、USD 应收），汇兑损益入 6604。
4. **试算平衡 → 财务报表**：核对 B/S 平衡、留存收益 = 期初 609,000 + 本期净利。
5. **结账**：FY 末（年结）才结转损益→留存；季结仅锁账。
6. **核对**：与客户提供的去年真实报表数字比对（待办 T7）。

> 备注：如需配套的「本期测试票据 / 银行流水」造数，可在此基础上生成与上表期初衔接的当期交易集（含 1 笔 USD 交易），用于解析 + 匹配 + 外币全链路验证。
