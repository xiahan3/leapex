# PRD · 会计报表模块（EasybookX）

> 模块：会计报表 Accounting Reports（账簿管理 之下）
> 适用：香港中小企业 SME · SME-FRS / HKFRS · 复式记账 · 权责发生制 · 币种 HKD
> 关联：[[财务报告生成规则与说明]]、[[PRD_账簿_v1.0]]、[[EasybookX_产品需求文档_PRD_v1.0]]
> 定位：账簿（底层账）→ **会计报表（按结账周期出具的报表中心）**；与做账流程中的【财务报表】关账出表步骤区分（见 §1.2）。

---

## 一、背景与定位

### 1.1 为什么要这个模块
- 账簿管理提供**底层法定账簿**（日记账 / 总账 / 科目余额表 / 往来明细 / 银行现金日记账 / 固定资产登记簿）。
- 用户需要一个**报表中心**：按**初始化设定的结账周期**（月 / 季 / 年）查看、对比、导出各类**会计报表**（损益表 / 资产负债表 / 现金流量表 / 权益变动表），并可回看**历史期间**。
- 本模块即「会计报表」，置于**账簿管理之下**，账套内（选定企业后可见）。

### 1.2 与【财务报表】的区分（重要）
| | 财务报表（做账流程内）| **会计报表（本模块）** |
|---|---|---|
| 位置 | 做账闭环：期末调整 → 试算 → **财务报表** | 账簿管理之下的**报表中心** |
| 用途 | 当期**关账出表**（确认财务报表 / 结转分录 / 出表前完整性校验）| 按**结账周期**查看 / 对比 / 导出**各期**会计报表 |
| 期间 | 当前账期 | **月 / 季 / 年**（历史 + 当前），随结账周期 |
| 报表 | P&L + B/S | **四表**：损益 / 资产负债 / 现金流量 / 权益变动 |
> 二者数据同源（同一总账 / COA），口径一致；财务报表偏「做账动作」，会计报表偏「查询出具」。

### 1.3 与账簿的关系
会计报表由**账簿（总账 / 科目余额表）按科目类别（COA 父子级）归集**得出：报表 = 期初 TB 余额 + 本期已确认发生额（权责发生制，仅 `confirmed`）。科目代码 / 名称 / 类别 / 正常余额唯一来源于**科目表 COA**（单一来源）。

## 二、香港会计报表体系（SME-FRS）

| 报表 | 英文 | 性质 | 香港要点 |
|---|---|---|---|
| **损益表** | Income Statement / P&L | 期间累计（流量）| 营业额→毛利→费用→除税前溢利→利得税→税后净利 |
| **资产负债表** | Statement of Financial Position | 期末时点（存量）| 资产 = 负债 + 权益；备抵科目净额列示 |
| **现金流量表** | Statement of Cash Flows | 期间累计 | 经营 / 投资 / 筹资；**SME-FRS 下小型私人公司可豁免** |
| **权益变动表** | Statement of Changes in Equity | 期间累计 | 期初留存 + 本期净利 − 股息 = 期末留存；含股本 / 折算储备 |
| 附注 | Notes | — | 法定财务报表含；管理报表可略 |

> 报表逐行**取数公式**见 [[财务报告生成规则与说明]] §6（父子级归集，不做报表公式配置）。

## 三、结账周期 × 报表期间 ★

报表期间随**初始化设定的结账周期**组织：

| 结账周期 | 报表期间（本模块可选）| 报表性质 |
|---|---|---|
| **月结 Monthly** | 各月（当月 + 历史月，可选本年累计 YTD）| 月报 = **管理报表** |
| **季结 Quarterly** | 各季 Q1–Q4（当季 + 历史季）| 季报 = **管理报表** |
| **年结 Annual** | 各财政年度（本财年 + 上年度）| 年报 = **法定财务报表** |

**口径（承 [[财务报告生成规则与说明]] §6.1）**：
- **月 / 季报 = 管理报表**：损益**不结转**（财年内累计）、**未经审计**；B/S 权益按「股本 + 期初留存(b/f) + 本财年累计净利（未结转）」列示。
- **年报 = 法定财务报表**：财年末**年结结转**（损益 → 3103 本年利润 → 3101 留存收益、清零）、待 CPA 审计、报税用。
- 即使按月结，仍可按 **月 / 季 / 年** 汇总查看（季 = 3 个月、年 = 12 个月）。

## 六、数据来源与生成规则
- **来源**：总账 / 科目余额表（由已确认分录 + 期末调整聚合），按 COA 父子级归集。
- **公式**：损益 `Σ(cat=I,cr−dr) − Σ(cat=X,dr−cr)`；资产 `Σ(cat=A,dr−cr)`；负债 `Σ(cat=L,cr−dr)`；权益 `Σ(cat=E,cr−dr)+本期损益`；现金流间接法；权益变动滚存。逐行见 [[财务报告生成规则与说明]] §6。
- **期间口径**：损益 / 现金流 / 权益变动 = 期间累计；资产负债表 = 期末时点。
- **外币**：B/S 期末汇率、P&L 平均价、折算差额入 3102（HKAS 21）。

## 七、状态说明

| 维度 | 状态值 | 含义 |
|---|---|---|
| 报表性质 | `management` 管理报表 / `statutory` 法定财务报表 | 月/季=管理，年=法定 |
| 出具状态 | `draft` 草稿（未关账）/ `issued` 已出具（已关账）| 关账后定稿 |
| 审计状态（年报）| `unaudited` 未审计 / `audited` 已审计 | 法定年报经 CPA 审计 |
| 平衡校验 | `balanced` / `unbalanced` | 资产=负债+权益 |

## 八、香港合规
- **SME-FRS** 简化披露；符合 Cap.622 S.359 报告豁免的小型私人公司适用。
- **法定年度财务报表须经 HKICPA 执业会计师审计**；本模块月 / 季报为**管理报表**，**不替代法定审计**（定位审前 / 管理）。
- 利得税两级 8.25% / 16.5%（年结正式计提）；香港不征 GST / VAT。
- 纪录 7 年留存为**企业（纳税人）义务**；平台存档为便利、不代担责。

## 九、异常处理
| 异常 | 处理 |
|---|---|
| 试算未平衡 | 报表标「草稿」，提示先平账 |
| B/S 不平衡 | 提示核对本期损益是否并入权益、科目类别、期初余额 |
| 选历史期间但未关账 | 标「草稿」，可看不可定稿 |
| 现金流量表豁免 | 小型公司可隐藏 / 标注「按 SME-FRS 豁免」|
| 报表代码与 COA 不符 | 提示以科目表为准（§1.3）|

## 十、与其他模块关系
- **账簿管理**：上游底层账（本模块在其下方）。
